# WiseTransact – Brand Guidelines & Design System

Figma Make Visual Reference Sheet v1.0 | May 2026

## Project Overview

WiseTransact is a mobile application that helps individuals build a clean, categorized record of their financial transactions through the year, so that at tax time they can hand a CA a comprehensive view of their inflows, outflows, and tax-relevant categorizations without manually building an Excel sheet from scratch.

### Headings (Inter)

* H1: Inter Semibold 600 • 24pt/32pt – Screen Titles
* H2: Inter Semibold 600 • 20pt/28pt – Card Titles
* H3: Inter Medium 500 • 18pt/24pt – Subsection Headers

### Body Text & Utility (Inter)

* Body Large: Inter Regular 400 • 16pt/24pt – Primary content
* Body Medium: Inter Regular 400 • 14pt/20pt – Secondary content
* Label: Inter Medium 500 • 12pt/16pt – Form labels, tags
* Small: Inter Regular 400 • 12pt/16pt – Metadata, timestamps, and secondary information

### Financial Numbers (JetBrains Mono)

* Amount Large: JetBrains Mono Semibold 600 • 32pt/40pt • Tabular – e.g. $24,567.89
* Amount Medium: JetBrains Mono Medium 500 • 20pt/28pt • Tabular – e.g. $1,234.56
* Amount Regular: JetBrains Mono Regular 400 • 16pt/24pt • Tabular – e.g. -$450.00 (negative shown in Rose)

## Complete Brand Guidelines Package

The brand guidelines were developed using AI (Figma Make, Version 2) and contain a 2000+ line comprehensive document. The design combines professional financial trust (Emerald green) with clarity and approachability.

* Color System – Primary (Emerald), secondary palettes (Amber, Rose, Sky, Violet), category colors, gradients, and accessibility standards
* Typography – Complete type scale with Inter for UI and JetBrains Mono for financial numbers, including specific sizing for displays, headings, body text, and amounts
* UI Components – Detailed specs for buttons (primary, secondary, tertiary, destructive, FAB), input fields, cards, lists, navigation, modals, tags, and progress indicators
* Charts & Data Visualization – Bar, line, and donut charts optimized for financial data
* Iconography – Complete icon system with categories, sizes, and container styles
* Spacing & Layout – 4pt grid system, touch targets, safe areas
* Motion & Transitions – Animation principles, timing, easing functions, and specific transition patterns for every interaction
* Illustration Style – Empty states, spot illustrations, data states
* Accessibility – Contrast requirements, screen reader support, reduced motion
* Implementation Notes – Design tokens, dark mode prep, performance guidelines
